# カフェ憩いのぜんさん ウェブサイト

神奈川県座間市にあるカフェ憩いのぜんさんの公式ウェブサイトです。

## プロジェクト構成

```
cafe-zensan/
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── main.js        # JavaScriptファイル
├── images/            # 画像ファイル用ディレクトリ
└── README.md          # このファイル
```

## 特徴

- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- スムーススクロールナビゲーション
- モバイル対応メニュー
- カフェらしい温かみのあるデザイン

## 店舗情報

- **店名**: カフェ憩いのぜんさん
- **所在地**: 神奈川県座間市栗原2706-8
- **アクセス**: さがみ野駅 徒歩20分
- **営業時間**: 火曜日～月曜日 11:00～16:00
- **定休日**: 年中無休

## コンセプト

のんびりしたい方が遊びに来てほしいカフェ。カラオケも楽しめる憩いの空間です。

## 使用方法

1. `index.html` をブラウザで開いて表示を確認
2. 画像ファイルは `images/` ディレクトリに配置
3. 必要に応じてコンテンツを編集
